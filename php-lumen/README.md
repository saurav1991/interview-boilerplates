## Install
- Install php > 7.1
- Install composer using
```sh
curl -sS https://getcomposer.org/installer | php
```
- Install all dependencies with
```sh
php composer.phar install
```

## Usage
Run the server with
```sh
php -S localhost:8000 -t public
```

## Tests
Run unit tests with
```sh
./vendor/bin/phpunit
```
